# Automation

Automation scripts for everyday macOS workflows.

## Includes

- `login/` — scripts that run at login
- `workflows/` — repeatable task flows
- `shortcuts/` — small time-saving helpers

Notable script:

- `login/mqlogin.sh` — stylish session boot for `macos-scripts` with `mqlaunch` integration

## Goal

Remove friction and automate the boring parts.
