# macOS Terminal Guide

A clean, searchable local HTML reference for common macOS Terminal commands.

## Overview

This folder contains a standalone HTML guide for macOS Terminal usage:

- searchable command reference
- categorized command groups
- practical examples
- clean visual layout for quick lookup

File included:

- `mac-terminal-guide.html`

## Purpose

The goal of this guide is to make common Terminal commands easier to browse and understand without needing to search the web every time.

It is designed as a fast local reference for:

- navigation
- files and folders
- search
- permissions
- processes
- networking
- package tools
- system-related commands
